#  Sistema de Shopping em Java

Este é um projeto simples em Java, feito para praticar lógica de programação e conceitos básicos de orientação a objetos.

O sistema simula o funcionamento de um shopping, permitindo:

- Criar um shopping com várias lojas
- Adicionar lojas ao shopping
- Listar as lojas cadastradas
- Buscar uma loja pelo nome

---

##  Tecnologias usadas

- Java
- Programação Orientada a Objetos (POO)

---

##  O que eu aprendi com este projeto

- Criar e usar classes em Java
- Trabalhar com arrays e laços de repetição
- Organizar o código em métodos
- Separar responsabilidades entre classes
- Pensar na lógica antes de programar

---

##  Como executar o projeto

1. Baixe ou clone o repositório:

2. Abra o projeto em uma IDE (IntelliJ, Eclipse, VSCode)

3. Execute a classe `Main.java`

---

##  Estrutura do projeto

- `Main.java` → Classe principal que inicia o sistema  
- `Shopping.java` → Classe que representa o shopping  
- `Loja.java` → Classe que representa uma loja
